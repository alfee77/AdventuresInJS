The switch statement

The switch statement is useful as a more elegant solutuion to multiple nested ifs. The following function is attached to the switch button. In the following example, the user is prompted for a number, and depending on the number entered, the code execution is switched. Note the use of break after the expression in each case X: statement. If break is not stipulated, code execution continues. This effectively means that case statements can be grouped (see case 2 and case 3).

function switchDemo(){ let a = +prompt('Input 0, 1, 2 or 3: ', ''); switch(a) { case 0: alert (0); break; case 1: alert(1); break; case 2: case 3: alert('2,3'); break; } }
