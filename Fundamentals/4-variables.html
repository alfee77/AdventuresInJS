Variables

This is a really basic task that shows how a variable can be created and assigned a value using the let keyword. The code declares two variables: name and admin. The string "John" is assigned to the variable name, and then the contents of the variable name are assigned to the variable admin. The call to the alert() function then displays the contents of the variable admin:

'use strict' let admin; let name; name = "John"; admin = name; alert(`"${admin}" has been assigned to the variable "admin"');
