2-Code Structure

Appending script in <script> </script> tags

The purpose here is to show what code structure looks like. This page represents a means of calling JavaScript code embedded within the same html page. This code can be found in the <script> </script> after the </body> tag:


    </body>

    <script>

        function showAnAlert(){ 
            alert("I'm JavaScript"); 
            document.getElementById("demo").innerHTML = "I'm JavaScript";
        }

    </script>

Embedding code in an external .js file

This takes exactlly the same code segment, but embeds it in an external .js file. This code can be found in the file referenced in the src of the <script> </script> tags after the </body> tag:


    </body> 
    <script src="js/taskShowanAlerttextJS.js"></script>

The function showAnAlert() can now be found embedded in the taskShowanAlertextJS.js file.


    function showAnAlert(){ 
        alert("I'm JavaScript"); 
        document.getElementById("demo").innerHTML = "I'm JavaScript";
    }
