Logical Operators

Logical AND: &&

This is all about logical operators. Run the code attached to the following button to see the logical && operator in action. The code prompts the user for their age and checks if it is NOT greater than or equal to 14 AND NOT less than or equal to 90 (it also uses the ! operator in the conditional check of the if... statement).

If the condition returns truthy, the statement 'The age ${age} is NOT between 14 and 90` is evaluated and posted to the page. If the condition returns falsy the statement `The age ${age} is between 14 and 90` is evaluated and posted

function checkAgeAnd(){ let age = prompt("Enter your age: ", 0); if (!(age >= 14 && age <= 90)){ document.getElementById("demo").innerHTML = `The age ${age} is NOT between 14 and 90`; } else{ document.getElementById("demo").innerHTML = `The age ${age} is between 14 and 90`; } }

If the condition returns truthy, the statement 'The age ${age} is NOT between 14 and 90` is evaluated and posted to the page. If the condition returns falsy the statement `The age ${age} is between 14 and 90` is evaluated and posted to the page.

function checkAgeOr(){ let age = prompt("Enter you age: ", 0); if (age < 14 || age > 90){ document.getElementById("demo1").innerHTML = `The age ${age} is NOT between 14 and 90`; } else{ document.getElementById("demo1").innerHTML = `The age ${age} is between 14 and 90`; } }
